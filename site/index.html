SharedWorld

Cloud-backed world handoff for Minecraft friends.

SharedWorld keeps world continuity simple for small groups. The active player hosts locally, world state is preserved through cloud-backed snapshots, and reconnects happen through e4mc instead of a permanent dedicated server.

Privacy Policy

Current Release

The current release line targets Fabric on Minecraft 1.21.11. The repository layout is already prepared for future loader expansion.

Hosting Model

SharedWorld includes a public backend codebase for self-hosting while also supporting an official hosted service.
